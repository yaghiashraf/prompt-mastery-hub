WORKFLOW AUTOMATION

CI/CD Automated Code Reviewer

Don't just run lints. Use the DEPTH Engine to spin up a simulated "Security Auditor" agent that reviews every Pull Request for logic flaws, not just syntax.

  • Dynamic Persona: Automatically assigns "Security Engineer" role.
  • Format Control: Enforces JSON output for easy parsing in GitHub Actions.
  • Chain of Thought: Forces "Attacker Mindset" simulation.
.github/workflows/audit.py 
import openai
from prompt_engine import generate_depth_prompt

# 1. Generate the Dynamic Prompt
system_prompt = generate_depth_prompt(
    role="Senior Security Engineer",
    task="Audit this Python code for injection vulnerabilities",
    context=pr_diff_content,
    framework="DEPTH",
    format="JSON"
)

# Output: "Act as a panel including a Penetration Tester..."

# 2. Call LLM
response = openai.ChatCompletion.create(
    model="gpt-4",
    messages=[
        {"role": "system", "content": system_prompt},
        {"role": "user", "content": "Analyze the following diff..."}
    ]
)

# 3. Block PR if Risk > High
if response.json()['risk_level'] == 'HIGH':
    block_pull_request(reason=response.json()['summary'])
