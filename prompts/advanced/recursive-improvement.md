# Recursive Self-Improvement

**Category:** Advanced
**Difficulty:** Intermediate
**Use Case:** Creative Writing, Critical Analysis, Code Refactoring

## The Template

### Step 1: The Draft
```markdown
[Insert your initial prompt or content here]
```

### Step 2: The Critique
```markdown
Review the content above. Act as a [Critic Role, e.g., Lead Editor / Security Auditor].
Critique the content based on the following criteria:
1. [Criterion 1, e.g., Clarity]
2. [Criterion 2, e.g., Security Vulnerabilities]
3. [Criterion 3, e.g., Engagement]

Provide a list of 3-5 specific, actionable improvements.
```

### Step 3: The Refinement
```markdown
Now, rewrite the content implementing all the improvements listed above.
Ensure the final output maintains the original [Core Meaning/Functionality].
```
