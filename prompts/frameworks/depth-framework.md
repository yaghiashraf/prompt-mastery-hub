# The DEPTH Framework

**Category:** Framework
**Difficulty:** Expert
**Use Case:** Complex Problem Solving, Strategic Planning

## The Template

```markdown
# DEPTH Prompting Framework

## 1. Define Perspectives
Act as a panel of experts including:
- [Expert 1, e.g., Data Scientist]
- [Expert 2, e.g., UX Researcher]
- [Expert 3, e.g., Business Strategist]

## 2. Establish Success Metrics
The output must achieve the following:
- [Metric 1, e.g., Reduce latency by 20%]
- [Metric 2, e.g., Increase readability score to 8th grade level]

## 3. Provide Context
Here is the background information:
[Insert detailed context, user constraints, or historical data]

## 4. Task Breakdown
Execute the following steps sequentially:
1. Analyze the problem from each expert's perspective.
2. Synthesize the insights into a unified strategy.
3. Outline an execution plan.

## 5. Human Feedback Loop
Before generating the final plan, rate your confidence (0-100%) in achieving the success metrics based on the provided context. If confidence is below 80%, ask clarifying questions.
```
