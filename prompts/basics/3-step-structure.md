# The 3-Step Prompt Structure

**Category:** Basics
**Difficulty:** Beginner
**Use Case:** General Purpose, Writing, Coding

## The Template

```markdown
### 1. Context / Role
Act as a [Specific Role, e.g., Senior Copywriter / Python Expert].
Your goal is to [Specific Goal, e.g., write a landing page headline / debug this function].

### 2. Task Description
Please [Action] the following [Input Data].
Ensure you focus on [Specific Focus Area].

### 3. Constraints & Format
- Tone: [Tone, e.g., Professional, Witty]
- Format: [Format, e.g., JSON, Bullet Points, Markdown Table]
- Length: [Constraint, e.g., Under 280 characters]
- Examples:
  Input: [Example Input]
  Output: [Example Output]
```

## Example Usage

> **Act as a Senior Copywriter.**
> Your goal is to write a landing page headline for a new coffee brand.
>
> **Please write 5 headline variations** for the following audience: Busy tech professionals who work late.
> Focus on energy and clarity.
>
> **Constraints:**
> - Tone: Punchy, modern.
> - Format: Bulleted list.
> - Example Style: "Fuel your code. Not your anxiety."
